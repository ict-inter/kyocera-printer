# Install Kyocera printer

## For Intune
- Use `Install-Kyoceraprinter.ps1`.

## For manual installation
1. Download and move `Install-Kyoceraprinter-manually.ps1` to desktop.
2. Download and unzip KyoceraDriver.zip to `C:\`. Files should appear in `C:\KyoceraDriver`.
3. Open Powershell as administrator.
    1. Navigate to Desktop.
    2. Paste this command: `.\Install-Kyoceraprinter-manually.ps1` and hit enter.
4. Wait for script to finish and hit enter.
5. Printer should've been added successfully.
